# 🚀 CryptoPulse Dashboard

A modern and responsive Cryptocurrency Dashboard built using **HTML, CSS, JavaScript, API Integration, and React.js**.  
This project fetches real-time cryptocurrency market data and displays prices, market cap, daily trends, and search functionality.

---

##  Project Title

**CryptoPulse Dashboard**

---

##  Features

- Live Cryptocurrency Prices  
- Real-Time API Data Fetching  
- Search Coins Instantly  
- Responsive UI Design  
- Clean Modern Dashboard  
- Built with HTML, CSS, JavaScript  
- React.js Version Included  
- Easy to Customize  
- Beginner Friendly Project  
- Perfect for Resume / Portfolio

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- REST API  
- React.js  
- CoinGecko API
