# 🚀 CryptoPulse Dashboard

A modern and responsive Cryptocurrency Dashboard built using **HTML, CSS, JavaScript, API Integration, and React.js**.  
This project fetches real-time cryptocurrency market data and displays prices, market cap, daily trends, and search functionality.

---

##  Project Title

**CryptoPulse Dashboard**

---

##  Features

- Live Cryptocurrency Prices  
- Real-Time API Data Fetching  
- Search Coins Instantly  
- Responsive UI Design  
- Clean Modern Dashboard  
- Built with HTML, CSS, JavaScript  
- React.js Version Included  
- Easy to Customize  
- Beginner Friendly Project  
- Perfect for Resume / Portfolio

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- REST API  
- React.js  
- CoinGecko API

```
https://api.coingecko.com/api/v3/
```
EndPoint
```
https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd&order=market_cap_desc&per_page=20&page=1&sparkline=false
```
## ⚛️ How to Run React Version
Step 1:
```
Create React App
npx create-react-app crypto-dashboard
```
Step 2:

```
Replace src files with provided code.
```

Step 3:
```
Install Dependencies
npm install
```
Step 4:
```
Start Project
npm start
```
